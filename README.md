# description
这是一个关于化学实验室透明玻璃仪器的实例分割算法的研究，该项研究基于百度飞桨深度学习框架，可以实现对14类常见的实验透明玻璃仪器进行图像分割。
ppdet/modeling/AsppSolov2和ppdet/modeling/TransSolo是我们的工作成果。
